A scripting language is a programming language that can be interpreted by another program at runtime instead of being compiled by the computer's processor. Generally, scripting languages are easier to learn as a beginner rather than other programming languages such as C or C++. During runtime, scripting languages are usually converted to machine code by a program called interpreter.

Scripting languages can be used with HTML to add functionality to a webpage and make it more dynamic. An advantage of using scripting languages is that they allow complicated tasks to be executed in relatively few steps. Examples of scripting languages include: AppleScript, bash, Bourne shell, C shell, JavaScript, Korn shell, LotusScript, Perl, PowerShell, PHP, Python and Ruby.
