We used a couple of bash scripts to add and accelerate the creation of this websites. Using bash scripts allowed us to quickly insert html that is present on ever page such as the footer and navigation bar. We also used a bash script to quickly set the correct navigation bar element to have the active class so the right section of the navigation bar was highlighted. We also use a very small amount of javascript to power the google maps section on the homepage. This javascript calls the google maps api to generate the interactive map on the homepage. We could have used more javascript in our website however it would have added much more complexity which was beyond the scope of this project.
