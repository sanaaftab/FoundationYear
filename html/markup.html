A markup language is a language designed for processing text with the use of tags which define elements. Usually the tags are human readable; they use words that describe their command. The three main types of markup languages are: presentational markup, descriptive markup and procedural markup. Presentational markup refers to the use of margins and indentation. Procedural markup consists of programming constructs; it provides commands to a program which then creates presentational markup. Descriptive markup is more to do with the structure of a document rather than its appearance.

Some examples of markup languages include: HTML (Hypertext Markup Language), XHTML (eXtensible Hypertext Markup Language), XML (Extensible Markup Language), GML (Generalized Markup Language), SGML (Standard Generalized Markup Language) and LaTeX. HTML is the most universal and widely used markup language. It is a language used for creating web pages and the content of each webpage is defined by tags.
